# Manteclaw Skills Marketplace

10 production-ready OpenClaw skills for Base L2 DeFi, automated mining, agent health monitoring, and marketplace operations.

## Skills

| Skill | What it does | Install |
|-------|-------------|---------|
| **base-l2-agent-kit** | MCP server with 9 Base L2 DeFi tools (wallet, swaps, liquidity, price feeds, flash loans) | `/learn @manteclaw/skills-marketplace --skill base-l2-agent-kit` |
| **base-l2-automation** | Litcoiin mining + Bankr wallet ops + treasury automation | `/learn @manteclaw/skills-marketplace --skill base-l2-automation` |
| **gas-price-predictor** | Predict optimal gas prices on Base L2 using mempool analysis | `/learn @manteclaw/skills-marketplace --skill gas-price-predictor` |
| **litcoiin-miner** | Automated proof-of-comprehension mining on Base L2 | `/learn @manteclaw/skills-marketplace --skill litcoiin-miner` |
| **mev-protection-scanner** | Detect sandwich/frontrun attacks, suggest protection strategies | `/learn @manteclaw/skills-marketplace --skill mev-protection-scanner` |
| **agent-health-monitor** | Full-stack health monitoring for AI agent infrastructure | `/learn @manteclaw/skills-marketplace --skill agent-health-monitor` |
| **auto-marketplace-submitter** | Auto-submits skills to multiple marketplaces | `/learn @manteclaw/skills-marketplace --skill auto-marketplace-submitter` |
| **kimiim** | Kimi Group Chat integration and session management | `/learn @manteclaw/skills-marketplace --skill kimiim` |
| **time-awareness** | Agent temporal awareness utilities | `/learn @manteclaw/skills-marketplace --skill time-awareness` |
| **worker-safety** | Safety guardrails for autonomous agents | `/learn @manteclaw/skills-marketplace --skill worker-safety` |

## Install All

```bash
# agentskill.sh
/learn @manteclaw/skills-marketplace

# skills.sh
npx skills add manteclaw/skills-marketplace --yes

# ClawHub
clawhub install manteclaw--base-l2-agent-kit
clawhub install manteclaw--litcoiin-miner
# ... etc
```
