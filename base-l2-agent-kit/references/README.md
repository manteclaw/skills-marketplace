A production-ready starter kit that takes an AI agent from zero to earning in under 30 minutes on [Base](https://base.org) — Coinbase's L2 with sub-cent gas fees.

## Quick Start

```bash
# 1. Clone
git clone https://github.com/manteclaw/base-agent-earning-stack.git
cd base-agent-earning-stack

# 2. Install
pip install -r requirements.txt

# 3. Configure
export BANKR_API_KEY="your_key"
export OPENROUTER_KEY="your_key"

# 4. Run auto-miner
python auto_miner.py

# 5. Start x402 service
python x402_service.py
```
